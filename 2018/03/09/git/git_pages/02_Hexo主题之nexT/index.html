02_Hexo主题之nexT

发表于 2018-03-09 | 分类于 git > pages

下载&安装

下载

从github上下载安装包
或者使用gitbash从github上clone

安装

解压之后，放在hexo路径下的theme文件夹下
将解压后的文件重命名为next（必须）
打开站点的配置文件＿config.yml
将主题theme的值改为next

部署

打开hexo路径下，打开gitbash
执行命令hexo clean
执行命令hexo g
打开本地服务hexo s 或者hexo server -p 8070（更改被占用的默认端口，改为8070）
浏览器打开localhost：8070；查看新的主题效果，符合预期，则可以提交github。
执行命令hexo d；完成提交github pages。
