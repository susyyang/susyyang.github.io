Hexo主题next之头像图片等

发表于 2018-03-09

设置 头像

修改 主题配置文件 

增加或修改字段avatar， 值设置成头像的链接地址。
其中，头像的链接地址可以是：
站内地址：/uploads/avatar.png
互联网地址：http://example.com/avatar.png

设置背景动画

NexT 自带两种背景动画效果

canvas_nest
和
three_waves
只能开启其中一种

配置主题配置文件

# canvas_nest
canvas_nest: true //开启动画
canvas_nest: false //关闭动画

# three_waves
three_waves: true //开启动画
three_waves: false //关闭动画
