02_linux文件实践01（ll文件详细信息）

发表于 2017-10-03 | 分类于 remote_connection > linux

linux文件的详细信息包括

修改文件的所有者
命令：chowm （change owner 的缩写）
格式：chowm [-R] 用户user 文件目录
选项：-R 表示递归，目录修改时，需要使用-R选项

修改文件的所在的组
命令：chgrp（change group 的缩写）
格式：chgrp [-R] 所在组 文件目录
选项：-R 表示递归，目录修改时，需要使用-R选项
