02_Java服务器容器tomcat（配置）

发表于 2017-12-13 | 分类于 JavaEE > tomcat

windows独立环境下

下载：从tomcat官网下载tomcat最新版本的tomcat。
一旦下载了tomcat（tomcat包是免安装版本的），解压到方便的位置。例如，在您的windows系统中，则可以解压在C:\Apache Tomcat v7.0。
可选步骤：在windows环境变量里设置环境变量。

windows系统，独立环境下，Tomcat服务器启动方式：

  • 进入tomcat目录，双击startup.bat。mac平台双击startup.sh。
  • 进入控制台窗口，进入tomcat的bin目录下，运行startup.bat文件。
  • 验证：浏览器输入：http://localhost:8080/

windows系统，独立环境下，Tomcat服务器停止方式

  • 关闭tomcat运行窗口。
  • tomcat下bin目录，运行shutdown.bat文件。

设置ClASSPATH：

  • 由于Servlet不是Java平台的标准组成部分，所以您需要为编译器指定Servlet类的路径。
  • 点击：计算机—>属性—>高级—>环境变量—>CLASSPATH，设置CLASSPATH的值。
  • 设置成：tomcat安装目录（D:\Apache Tomcat v7.0\lib\servlet-api.jar）

eclipse开发工具中，tomcat环境搭建

已经配置好jdk。
已经安装好eclipse，能够正确运行。
进入：eclipse—>preferences—>server—>runtime environment—>add—>选择tomcat版本。这里需要注意的是tomcat版本和javaEE版本是否支持。
选择next：
输入tomcat需要设置的名字（name）；
选择tomcat服务器的安装目录，这里选择的是已经在windows系统在独立安装好的tomcat路径。（当然，我们也可以在eclipse中临时下载和安装，参照下面一个步骤）
可选步骤：eclipse中下载安装tomcat，点击download and install
选择JRE：选择工作平台默认的JRE就可以了。
点击finish，配置完成。

eclipse项目中，引用设置好的tomcat服务器。

Dynamic web project

  • 新建项目，点击target runtime，选择需要设置的tomcat服务器。
