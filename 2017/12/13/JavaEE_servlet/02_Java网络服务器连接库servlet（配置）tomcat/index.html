02_Java网络服务器连接库servlet（配置）tomcat

发表于 2017-12-13 | 分类于 JavaEE > servlet

安装tomcat

本机安装tomcat。（默认端口8080）
浏览器验证安装成功与否。（localhost:8080）

eclipse配置tomcat容器

  • eclipse切换到JavaEE环境下。
  • 菜单栏windows—preferences—server—runtime environments
  • add添加本机tomcat的安装路径。
    name：表示这个tomcat服务器的名字，可以自定义，也可以默认
    tomcat installation directory：表示配置的本机tomcat安装路径（D:\Apache Tomcat v7.0）
    JRE：表示运行时的java运行环境jre。

servlet支持库（包文件）

  • 打开tomcat安装目录。
  • 找到lib文件夹目录。
  • 里面包含很多jar包文件。其中servlet-api.jar就是servlet支持库。用的时候拷贝到项目中去。
