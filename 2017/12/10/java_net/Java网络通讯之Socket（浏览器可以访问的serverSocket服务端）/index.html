Java网络通讯之Socket（浏览器可以访问的serverSocket服务端）

发表于 2017-12-10 | 分类于 Java > net

socket类理解

  • 套接字，ip+port
  • 可以存储数据（水泵，往里面灌水）
  • 可以取出数据（水龙头，打开接水）
  • socket可以理解为一辆车。
  • 存储数据outputstream（车的上货区域）
  • 读取数据inputstream（车的卸货区域）
  • 面向对象的思维。
